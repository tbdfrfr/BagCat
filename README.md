# BagCat

[![forthebadge](https://forthebadge.com/api/badges/community/019b4b6c-a55a-74d3-997a-8cdb89f5d7d6.svg)](https://forthebadge.com)
BagCat is a React + Vite web app that provides a browser-like “hub” experience: search, quick links, an in-app tabbed viewer, and an app/game player UI. It ships with a Node/Fastify server that serves the built frontend and provides proxy-related endpoints used by the app.
